## Documento de Visão

### Nome do Projeto: TutorIA

**Propósito do Projeto**: Criar uma um assistente de IA que responda dúvidas de alunos baseado no conteúdo de uma disciplina no SIGAA.

**Objetivos**: Criar uma aplicação RAG (Retrieval-Assisted Generation) que possa fazer o webscraping dos conteúdos de uma disciplina, salvar os dados, e responder perguntas com base no material salvo definido pelos professores. O formato do app (site, extensão, app desktop/mobile, etc) ainda está a ser definido.

**Justificativa**: Para auxiliar no esclarescimento de dúvidas dos alunos.

**Benefícios**: Menor volume de dúvidas para o professor e monitores, ajuda no processo de estudo dos alunos.

**Envolvidos (Stakeholders)**:
* Alunos
* Professores
* Monitores
